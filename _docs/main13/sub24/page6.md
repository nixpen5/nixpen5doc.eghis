## 기초 수가

- 차트 실행 시 2024년 기초 수가 등록 툴 자동 실행
- 자동 설치 목록
    - 2024년 기초 수가 등록
    - 2024년 약국 수가 등록 (점수당 단가 반영)
    - 2024년 공휴일 등록
- 기초 수가 등록 툴 자동 실행 화면
    - 대기시간 경과 후 설치가 진행 됩니다.
    - 병원내 1대의 PC 에서 설치가 완료되면 추가로 실행되지 않습니다.
